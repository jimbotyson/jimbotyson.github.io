Do Files and Log Files

Learning in this topic

The objective of this topic is to learn about two non-data filetypes

  • do files
  • log files

You will understand how to create each and what their proper uses are and you will be able to summarize the advantages of using a do file over working interactively at the command line.

The essential activities in this topic are watching the video and trying out the do file example code below.

The do file

Do files provide a simple way to batch together Stata commands and to automate Stata procedures.  A do file is a simple text file that contains Stata commands, usually one to a line, and possibly simple programming constructs such as loops (to repeat actions) and conditionals (to make decisions or selections).

It is very convenient to work on the Stata command line, but there are also advantages to creating do files, especially for more complicated procedures.  Using a do file

  • helps you create work is reproducible
  • makes it much  easier to check and debug your procedures  than by scrolling through your command history
  • if you create a do file that does the data preparation and the analysis, you are sure that you are always starting with the correct data

The log file

A log file is an automatically created record of the input and output of a stata session.  It comtains the commands you have issued to Stata and Stata's responses.  There are two formats for log smcl (pronounced smickle) and plain text.  SMCL is a rich mark up format that can be used by sophisticated users to produce nicely formatted output.  In general, a plain text log file is adequate.

You start a log file at any time with the command

log using "filename_for_my_log", text

and you close and save with the command

log close

which will also save the file.

A log file is a useful record of what you do in a session.  I strongly recommend that logfiles should have names including the project name and the date of the Stata session.  If you regularly use a standard format for log file names you will make your life much easier.
