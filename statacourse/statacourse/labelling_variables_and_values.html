Labelling variables and values

Learning in this topic

The essential activity in this topic is watching the video linked below.

The objectives of this topic are to explain how both variables and values can be labelled.  A variable in a dataset may have a name that isn't very human friendly and while we would not normally want to change the name (it may for example be the standard for a well known data set or family of data sets) nonetheless it is helpful if we can use a friendly name in output such as tables and graphs.  A value for some variable may also be labelled.  For example, we may have a variable that ranks cases based on the intervals into which the scores fall.  We might wish to label these intervales, perhaps low, mid and high.

Labels

Please watch this video.

You can label a variable or its values (if they are discrete).

If for example you have questionnaire data, you may have variables named

q1, q2,...,q100

This is convenient in some respects but it is obviously difficult to remember the actual content of each of these.  It makes sense to label the variables with the actual wording of the questions.

Similarly it is useful to have human friendly labels for the discrete levels of categorical variables so that if a variable control had values 0 and 1 we could label them treatment and control.

Labelling a variable

You can label a variable with the command

label variable "I chose this label"

Define a label set for values

A factor variable is a categorical variable (nominal or ordinal).  As we saw when the bysort prefix command was introduced, factors can be used to group cases of data.  For example, the factor sex may divide your data into a number of subsets.

Often, factors are encoded as numbers, 1/2 for male or female or 0/1 for yes or no.  Often it is convenient to have a human friendly label for these factor levels.  We create labels with

label define labelsetname value label...value label

For example

label define classes 1 "First Class" 2 "Second Class" 3 "Third Class"

This creates a new set of labels called classes such that a case with value 1 will be labelled First Class, 2 Second Class and 3 Third Class.  This set of labels can now be applied to any factor variable - nominal or ordinal - with three levels and numeric values 1,2 and 3.

Apply a label set to variable values

The label set is applied to the values of a factor with the following command

label values variable labelsetname

For example

label values class classes
