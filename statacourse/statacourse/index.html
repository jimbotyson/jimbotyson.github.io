Practical Stata Home

In the following learning package some topics contain a link to a video that explains some Stata functionality, guidance on Stata commands and functions and perhaps PDF material to read.  For some topics there may also be activities for you to explore on your own, quizzes that give instant feedback and exercises for you to practise using Stata.

Before attending the classroom Stata session, you must complete all the topics.  You will also find it useful to take notes.

If in completing a topic you create a script, you should ensure that you put commentary in the script so that you can understand it later.

You can use stata on UCL managed machines in public spaces or on the UCL remote desktop when you have an internet connection.  You do not need to download or install Stata to complete this learning.

You navigate topics using the expandable tree to the left of the content.

Objectives

Aim:

To be able to undertake elementary data management and statistical analyses in Stata

Objectives:

After taking this course you will be able to

  • Identify major elements of the Stata interface
  • Import data in some common formats
  • Compute new variables
  • Add labels to a dataset to make it self-documenting
  • Recode variables and derive new variables from existing ones
  • Produce summary statistics, tables and graphs
  • Find online help about a given topic or command
  • Keep a record of you work and ensure it is reproducible

Are you ready to go?

Pre-requisite knowledge

This course assumes that you have some knowledge and skills before you start.  For example, it assumes that you know some statistics:

  • descriptive statistics such as measures of
    • location
    • dispersion
    • shape
  • the use of graphs and plots
  • the different kinds of data that are analysed statistically such as
    • continuous versus discrete data
    • ordinal versus ration data (and the other levels of measurement)
  • an understanding of hypothesis testing, significance and confidence intervals

Digital Capabilities

It also assumes that you have basic computing skills such as file management and using a text editor or word processing program.

Learning online

The course also assumes that you are comfortable with online learning.  What previous experience of online learning do you have?  How did you find any previous online learning experiences?  Did you enjoy them?  What are the challenges of on-line learning for you and what are the advantages?

Ready to go?

Take a moment to think about the objectives of the course and some of the assumptions outlined above.  Do you understand the objectives and do they correspond to what you want from the course? Do you have the kind of pre-knowledge that the course assumes?
