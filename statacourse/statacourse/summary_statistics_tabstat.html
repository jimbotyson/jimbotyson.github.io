Summary statistics: tabstat

Learning in this topic

The objective in this topic is to explain the use of the Stata command tabstat for producing summary statistics.

Please watch this video which explains the use of the tabstat command.

The tabstat command takes the general form

tabstat variable, statistics(...) by(factor)

With tabstat the following statistics are available

mean

mean

count

count of nonmissing observations

n

same as count

sum

sum

max

maximum

min

minimum

range

range = max - min

sd

standard deviation

variance

variance

cv

coefficient of variation (sd/mean)

semean

standard error of mean (sd/sqrt(n))

skewness

skewness

kurtosis

kurtosis

p1

1st percentile

p5

5th percentile

p10

10th percentile

p25

25th percentile

median

median (same as p50)

p50

50th percentile (same as median)

p75

75th percentile

p90

90th percentile

p95

95th percentile

p99

99th percentile

iqr

interquartile range = p75 - p25

q

equivalent to specifying p25 p50 p75

Tabstat (and understanding code)

Create a file with the following code lines.

import excel "http://www.ucl.ac.uk/~ccaajim/results.xls", clear first case(lower)

gen avxm=(maths+english+history)/3

local conts maths english history avxm

format `conts' %9.2f

tabstat `conts', statistics(mean, var, sd) by(sex)

Run the code and then comment each line explaining what it does.  Use the Stata help or any other resource to help you explain the code.

Feedback

*import data from an excel spreadhsheet, clear data from memory first, and put variable names from the first row in lower case

import excel "http://www.ucl.ac.uk/~ccaajim/results.xls", clear first case(lower)

*create a new variable avmx which is the average of the three exam scores

gen avxm=(maths+english+history)/3

*create a local macro (name) to refer to all the continuous variables conveniently

local conts maths english history avxm

*specify that the continuous variable output should be displayed to two decimal places

format `conts' %9.2f

*show a table of the mean, variance and standard deviation for all the continuous variables split by sex

tabstat `conts', statistics(mean, var, sd) by(sex)
