The Class Room Session

Congratulations

You have now completed the pre-learning for the Practical Stata course.

When you come to the classroom session, we will assume that you have completed this work.

The class room session involves work on a data set involving writing a data preparation do file; managing data; describing and exploring data; hypothesis testing and regression modelling.

Next steps

The classroom session will involve a range of statistical skills to be used in Stata.Â  You are not expected to have the Stata knowledge in advance of the classroom session but you are expected to have the knowledge of statistics.

Topics you will need to have covered before attending include:

Descriptive statistics

  • mean
  • median
  • mode
  • variance
  • standard deviation
  • correlation

Inferential statiistics

  • t-tests (independent and paired)
  • chi-square test
  • linear regression

You will not be able to follow that session if you do not have this statistical knowledge.
