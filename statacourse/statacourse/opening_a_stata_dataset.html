Opening a Stata Dataset

Learning in this topic

The objective of this topic is to learn to load Stata datasets whether focally or from a remote source.

Loading data from remote sources is an important technique and can be part of a strategy to treat data as read only.

The essential activity in this topic is watching the video material and then completing the activity to download a data set from a remote source.

Please watch this video which explains how to open and save Stata datasets.

The format of files with the extension .dta is the proprietary Stata format.  If you use the bare save command, this is the format that Stata will save in.

Downloading a Stata dataset from a web address

Using the documents attached below from Stata's help, load the data set results.dta which is stored on the website

http://www.ucl.ac.uk/~ccaajim

You will need to understand the commands webuse set and webuse.

Webuse set is used to tell Stata to use a specific url (web address) as the source of data files.

Webuse is used to download a data file from a website in to Stata memory.

The first command therefor, is followed by the website address.  The second command is followed by the name of the datafile.

A full description of each command is found the manual links below.

Stata Manual Chapter

Here are the Stata 13 manual chapter that describes opening and saving datasets and a page from the manual describing the webuse command.

Opening a remote dataset

Using the documents attached below from Stata's help, load the data set results.dta which is stored on the website

http://www.ucl.ac.uk/~ccaajim

You will need to understand the commands webuse set and webuse.  The recommended solution is in the feedback.

Feedback

First, redirect the webuse command to the correct source:

webuse set "http://www.ucl.ac.uk/~ccaajim"

Now you can use webuse to download the file

webuse results.dta

You can check that the data has been loaded by looking at the variables pane.
