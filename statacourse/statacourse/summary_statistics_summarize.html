Summary statistics: summarize

Objectives

The objective in this topic is to master some basic exploratory techniques for understanding data in Stata.

The essential activity in this topic is working through the UCLA material linked to below.

Please watchÂ  this video to learn about the summarize command.

Exploring Data

Now, please work through this web page about exploring data.

As you study the page, you should try out the commands discussed using the data set which you can download by issuing the command at the start of the notes:

use http://www.ats.ucla.edu/stat/data/hs0, clear

You may find it a useful exercise to create a do file containing the commands and experiment with running it.
