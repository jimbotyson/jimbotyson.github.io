Getting Started

Learning in this topic

The objectives of this topic are to become familiar with the Stata interface so as to be able to enter Stata commands, identify and examine variables and values and use the review window to access the command history.

Each of these elements is important in using Stata, even if - as is often the case - you go on to do most of your work by writing scripts to automate processes.

The essential activity in this topic is watching the video material.

Please watch this video, produced by the University of California San Francisco, which explains the basics of the Stata interface.

Stata Windows environment

The University of North Caroline has a one page guide to elements of the Stata interface that you may find useful.  You do not need to read it right away, but you should keep it on hand for reference when you are using Stata later.
