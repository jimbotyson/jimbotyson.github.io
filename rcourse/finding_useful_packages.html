Finding useful packages

Indicating the strength of association with categorical data

You have established - using Pearson's Chi Square Test - that two variables Music Genre Preference and Final Exam Grade are associated, and you would like to estimate the strength of the association. 

Reading Wikipedia suggests that you can use a statistic known as Cramér's V which will indicate the degree of correlation between the two.

Searching your basic RStudio installation doesn't provide a way to compute the statistic.

Install the package sos.  This package includes a function findFn() which will search package documentation pages in on-line repositories and list those that include your search term.

Use the findFn() function in the following format:

sos::findFn("")


to find a package that will compute Cramér's V for you.

Feedback

The search that worked for me was

sos::findFn("cramer")

The results open in a web browser window and in this case there were a lot of results - 113 packages!  Looking down the list on the page, I saw a package DescTools that calculates the statistic we are interested in.
