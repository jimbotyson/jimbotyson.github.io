Deriving new variables from existing data

We have a data frame results (download link below) that has six variables.  Ignoring the surname variable, we have:

Variable Type Level
class numeric nominal
gender numeric (binary) nominal
maths numeric ratio
english numeric ratio
history numeric ratio

You should use the R function str() and compare its output with this table.

The variables maths,english,history are examination scores out of 100. The class variable simply tells us which of three teachers taught the pupils. The gender variable is a binary female/male distinction with 1 = female and 2 = male.

To select a specific column (usually that is a variable) from a data frame, we use the column or component selector $.  In this data frame the column containing maths scores can be referred to by the expression results$maths.  This is a shorthand for writing a selection statement for all the rows in the fourth column.

We want to create and add a variable avxm which is the average of the three examination scores. We could run the following code:

avxm<-(results$maths+results$english+results$history)/3

and add it to the dataframe with

cbind(results,avxm)

Or we could achieve the same end with one line:

results$avxm <- (results$maths+results$english+results$history)/
