File Attachments

Please download these files for this task.  One is a file of data in a plain text format with comma separators.  The other is the data dictionary that describes the data items in the first.

  • PUMSDataDict13.pdf (New Window)

    A data selection challenge


    The file ss13hca.csv contains a single year subset of data from the American Community Survey Public Use Micro-data Sample which reports on characteristics of households in California.

    You will need the data dictionary in file PUMSDataDict13.pdf(see link below). This file will explain what data the columns that this file contains.

    You will need to find the variables that you need to examine for this task from the data dictionary.  You will probably want to use the find function in your pdf reader to search this data dictionary.

    Read the data file into an R data frame.

    From the complete data frame create a new data frame containing data meeting only these constraints:

    1. Only include Housing units.
    2. Only include households with a dwelling on a less than one acre plot.
    3. Only include households with children.
    4. Only include households with family income between $75,000 and $150,000

    Choose a suitable name for this data frame.

    How many cases are there that meet these conditions?

    Create a vector of the income of those households where no child is under six years old.

    Feedback

    The relevant variables are TYPE, FES, FPARC, ACR.

    The function nrow() can tell you the number of rows in a data set.
